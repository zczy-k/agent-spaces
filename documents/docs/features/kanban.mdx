# Kanban 看板

Agent Spaces 内置 Kanban 看板系统，提供拖拽式任务管理。每个工作空间拥有独立的看板。

## 功能概览

- **多列看板** — 创建多个列（如"待办"、"进行中"、"已完成"），自由组织工作流
- **拖拽排序** — 基于 @dnd-kit 实现的拖拽操作，支持列内排序和跨列移动
- **优先级筛选** — 按任务优先级过滤显示
- **搜索过滤** — 关键词搜索快速定位任务
- **布局切换** — 支持水平和垂直两种看板布局

## 创建看板

1. 在工作空间中打开看板面板
2. 系统自动为工作空间创建默认看板
3. 可自定义列名称和顺序

## 列管理

- **添加列** — 在看板中创建新的列
- **重命名列** — 修改列名称
- **删除列** — 移除不需要的列（列中的任务会移至相邻列）
- **排序** — 拖拽调整列的显示顺序

## 任务管理

### 创建任务

- 点击列底部的「添加任务」按钮
- 填写任务标题和描述
- 设置优先级（高/中/低）
- 保存后任务出现在对应列中

### 操作任务

- **拖拽移动** — 拖拽任务到不同列
- **排序** — 在列内拖拽调整任务顺序
- **编辑** — 修改任务内容和优先级
- **删除** — 移除已完成的任务

## 布局切换

看板支持两种布局模式：

- **水平布局** — 列从左到右排列（默认）
- **垂直布局** — 列从上到下排列，适合窄屏

通过工具栏的布局切换按钮一键切换。

## 数据存储

看板数据存储在本地 SQLite 数据库中：

- `kanban_boards` — 看板表
- `kanban_columns` — 列表
- `kanban_tasks` — 任务表

数据按工作空间隔离，不会上传到外部服务。
