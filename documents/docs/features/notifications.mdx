# 通知中心

通知中心将 Agent 的状态变更推送到你常用的通讯工具或设备，让你随时随地掌握项目进展。

## 支持的通知渠道

### 飞书（Lark）

通过飞书 Bot 推送通知：

1. 在飞书开放平台创建应用，获取 App ID 和 App Secret
2. 在项目设置中配置飞书通知
3. 扫码授权后，Agent 状态变更会自动推送到飞书群

### 企业微信（WeChat Work）

通过企业微信 Bot 推送通知：

1. 在企业微信管理后台创建应用
2. 在项目设置中配置企业微信通知
3. 扫码登录后，通知自动推送到企业微信

### Native 通知

支持 Tauri 桌面客户端和浏览器的原生通知：

- **Tauri** — 使用系统原生通知 API，支持 macOS、Windows、Linux
- **Browser** — 使用 Web Notification API，需用户授权

Native 通知无需额外配置，开箱即用。

## 通知内容

以下事件会触发通知：

- 议题状态变更（创建、进行中、完成、关闭）
- 任务执行进度更新
- Agent 执行完成或失败
- 代码审核结果

## Bot 斜杠命令

在飞书或企业微信中与 Bot 对话时，可以使用内置命令远程操控 Agent：

| 命令 | 说明 |
|------|------|
| `/status` | 查看当前议题和任务状态 |
| `/list` | 列出所有议题 |
| `/agents` | 查看已配置的 Agent 列表 |
| `/workspace` | 查看工作空间信息 |
| `/usage` | 查看 Token 用量统计 |

更多命令可在 Bot 对话中输入 `/help` 查看。

## 配置方式

通知配置在工作空间的「项目设置」面板中：

1. 进入工作空间
2. 打开项目设置
3. 切换到「通知配置」标签
4. 选择通知渠道并填写凭证
5. 保存配置
