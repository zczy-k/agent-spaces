# Agent Spaces

Agent Spaces 是一个**本地多 Agent 协同编程平台**。通过可视化 Workflow 编辑器编排 Agent 执行流程，或在频道聊天中 @mention Agent 直接触发执行，实现任务的自动分发、代码修改、审核与提交。

## 为什么选择 Agent Spaces？

- **完全本地运行** — 代码不离开你的机器，数据存储在本地，安全可控
- **可视化 Workflow 编排** — DAG 拓扑编辑器，拖拽 Agent 节点、连线定义依赖，灵活编排执行流程
- **多 Agent 协同** — agent、scheduler、task_creator、bot 四种内置角色 + 自定义角色，各司其职
- **IDE 级别体验** — 集成代码编辑器、终端、Git 操作、议题管理，一站式开发
- **多运行时支持** — Claude Code、OpenAI Codex、Open Agent SDK 三种 Agent 运行时
- **实时通知** — 通过飞书、企业微信、Native 通知接收 Agent 状态变更，远程操控
- **国际化** — 中英文自由切换，本地化体验

## 核心能力

| 功能 | 说明 |
|------|------|
| 工作空间管理 | 绑定本地代码目录，支持文件夹浏览和 Git Clone |
| Workflow 编辑器 | DAG 可视化编排，拖拽 Agent 节点、连线定义依赖，自动布局 |
| 代码编辑器 | 基于 Monaco Editor 的完整编辑体验 |
| 终端 | 集成终端，可直接在平台内执行命令 |
| 议题管理 | 创建 Issue 并选择 Workflow 模板，自动编排 Task 执行 |
| 频道聊天 | @mention Agent 触发对话和任务执行 |
| Git 操作 | 可视化分支管理、提交、合并，Commit Agent 自动生成提交信息 |
| Agent 编排 | 四种内置角色 + 自定义角色，Workflow 驱动自动化协作 |
| 通知中心 | 飞书/企业微信/Native 消息推送和远程命令 |
| 用量统计 | Token 消耗、费用估算、趋势图表、按模型统计 |
| LLM 管理 | 多模型配置、Anthropic Bridge 协议中转 |
| 认证系统 | Secret Key + Bearer Token 认证 |
