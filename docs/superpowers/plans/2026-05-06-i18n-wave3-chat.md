# i18n 实施计划 — 第 3 波 Part 1：Chat 组件

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将 chat 目录下 13 个组件的所有硬编码文本替换为 i18n 翻译调用。

**Architecture:** 每个组件添加 `useTranslations('chat')` hook，替换硬编码文本为 `t('key')` 调用。翻译 key 按组件子域划分（如 `chat.status.*`、`chat.input.*`、`chat.messageParts.*`）。需要在 en.json 和 zh.json 的 `chat` namespace 中新增约 61 个 key。

**Tech Stack:** next-intl useTranslations, React hooks

**Spec:** `docs/superpowers/specs/2026-05-06-i18n-design.md`
**Depends on:** `docs/superpowers/plans/2026-05-06-i18n-wave1-infrastructure.md` (Task 1-4 必须完成)
